# Bike-Sale-Dashboard- Data-Analysis (Interactive Dashboard Using MS-EXCEL)

### PROJECT OBJECTIVE
The objective of this project is to analyze customer data to understand the key factors influencing bike purchases.
By visualizing demographics such as age, gender, income, marital status, education, occupation, and commute distance,
the dashboard aims to help businesses identify potential customers, track sales trends, and make data-driven marketing decisions to boost bike sales.

## DATA SET USED
- <a href="https://github.com/Girianiket/Bike-Sales-Dashboard-Excel/blob/main/Data%20Set.xlsx">Dataset</a>

## QUESTIONS
1. Which gender has a higher average income among bike purchasers?
2. Does marital status influence bike purchasing decisions?
3. How does education level affect the likelihood of purchasing a bike?
4. Is there any relationship between the number of children and bike ownership?
5. Which occupation group (e.g., Management, Clerical, Manual) purchases the most bikes?
6. How does commute distance impact the decision to buy a bike?
7. Which age group (Young, Middle Age, Old) is most likely to purchase a bike?
8. Is there a significant difference in average income between those who purchased and those who didn’t?
9. What is the trend of bike purchases across different age ranges (e.g., 30–60 years)?
10. Which combination of demographic factors (e.g., Married + Graduate Degree + Management) shows the highest purchase rate?
